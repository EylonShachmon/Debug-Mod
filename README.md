# Debug-Mod
custom debug mod for Hollow Knight, until the API works again and normal debug can be used

To use this mod, you first have to set up BepinEx (https://github.com/BepInEx/BepInEx/releases/tag/v5.4.23.4) in your Hollow Knight folder. Do know that BepinEx does not work with the normal Modding API, so you will need to get a separate install of the game if you already got mods for your game.

Once BepinEx is installed and working, simply drop the .dll file into the Hollow Knight/BepInEx/plugins folder
